About This Project

Telegram Similar Channels Crawler

A tool to discover similar channels on Telegram and export them to Google Sheets

Purpose

This tool helps you discover new Telegram channels that are similar to the ones you already know and like. It's useful for:

  • Content creators looking for similar channels in their niche
  • Marketers researching potential promotion channels
  • Users who want to discover new content sources
  • Researchers analyzing channel networks
How It Works

The crawler leverages Telegram's channel recommendation algorithm to find similar channels. The process includes:

  1. Connecting to Telegram using your API credentials
  2. For each input channel, finding similar channels based on content and audience overlap
  3. Retrieving metadata like member count and channel info
  4. Exporting all found channels to Google Sheets for easy analysis
Technical Details

This application is built with the following technologies:

  • Backend: Python
  • Web Framework: Flask
  • Telegram API: Telethon
  • Google Integration: gspread
  • Authentication: oauth2client
  • Config Management: python-dotenv
  • Frontend: Bootstrap 5
  • Icons: Font Awesome
  • Database: None (file-based)
Important Notes
  • This tool requires your own Telegram API credentials to function properly
  • The first time you run the crawler, you'll need to authenticate with your Telegram account
  • Telegram has rate limits - processing many channels might take time
  • All data is exported to your own Google Sheet - no data is stored on our servers
  • For assistance setting up your credentials, visit the Configuration page
